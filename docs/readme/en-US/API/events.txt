<h2>events</h2>
<p>utils-where/events makes it possible to support &quot;longpress&quot; on mobile, which differs from contextmenu<br>
it simulates longpress by touchstart and touchend, <strong>only import it when the contextmenu doesn&#39;t work as u wish!</strong></p>
