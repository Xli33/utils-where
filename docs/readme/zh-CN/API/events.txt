<h2>events</h2>
<p><code>utils-where/events</code> 使得在移动设备上支持 &quot;longpress&quot; 成为可能，不同于 contextmenu<br>
它通过 touchstart 和 touchend 模拟长按，<strong>仅在 contextmenu 的行为不符合您的要求时才导入它！</strong></p>
